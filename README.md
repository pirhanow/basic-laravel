
 👨‍💻 О проекте
Это полнофункциональное веб-приложение для управления контентом, построенное на Laravel 11. Проект демонстрирует переход от стандартного MVC к более гибкой архитектуре с выделенным слоем бизнес-логики (Service Layer). 

Система включает в себя публичную часть и защищенную админ-панель на базе AdminLTE 3.

---

 🚀 Ключевой функционал
• Управление контентом (CRUD): Полный цикл создания, редактирования и удаления постов.
• Связи Many-to-Many: Реализована сложная связь между Постами и Категориями через промежуточную таблицу (Pivot table).
• Безопасность (Auth & Middleware): Админ-панель полностью защищена. Доступ к управлению контентом имеют только авторизованные пользователи.
• Мягкое удаление (Soft Deletes): Реализован механизм восстановления данных после удаления.
• Профессиональная Админка: Полная интеграция шаблона AdminLTE 3 для удобного управления системой.

---

 🏗 Архитектурные особенности
Проект написан с соблюдением принципов SOLID и DRY:
• Service Layer Pattern: Вся бизнес-логика вынесена из контроллеров в отдельные сервисы (app/Services), что делает код тестируемым и чистым.
• Form Requests Validation: Валидация данных инкапсулирована в специальные классы запросов, разделяя логику проверки и обработки.
• Blade Templating: Использование мастер-шаблонов и секций для исключения дублирования HTML-кода.
• Dependency Injection: Активное использование внедрения зависимостей через конструкторы классов.

---

 🛠 Стек технологий
- Backend: PHP 8.2+, Laravel 11
- Database: MySQL (Eloquent ORM)
- Frontend: Blade, AdminLTE 3 (Bootstrap 4)
- Dependency Manager: Composer

 📦 Установка и запуск

1. Клонируйте репозиторий:
   
   git clone https://github.com/pirhanow/basic-laravel.git
   cd basic-laravel
   

2. Установите зависимости:
   
   composer install
   

3. Настройте окружение:
   - Скопируйте .env.example в .env
   - Настройте подключение к БД в .env

4. Подготовьте приложение:
   
   php artisan key:generate
   php artisan migrate
   php artisan storage:link
   

5. Запустите локальный сервер:
   
   php artisan serve
